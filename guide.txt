FRONTEND CREATION
create a client folder and inside it, give commands in terminal
    npm create vite@latest  //specify the stack
    npm i
    npm run dev

    in src folder, create a "components" folder, create each component as a folder here


create a backend folder
    in terminal, give npm init. this will create a package.json file
    create a server.js file in backend folder
    next run  npm i express to install express
    import express in server.js and assign port number (3 lines of code)

    create a .env file. add your environment variables
    "npm i mongodb"
    "npm i dotenv"
